# 更新长期记忆 (Update Memory)

## 内容: $ARGUMENTS

## 两种模式

### 模式一：主动记录
当 `$ARGUMENTS` 有内容时，直接写入 MEMORY.md。

```
/update-memory 新增 wopal-admin 项目规划
```

### 模式二：提炼
当无参数时，扫描短期记忆，提炼精华。

```
/update-memory
```

---

## 提炼流程

1. **扫描**：读取 `memory/` 下近 7 天的短期记忆
2. **筛选**：提取 `[教训]`、`[决策]`、`[里程碑]` 标记
3. **提炼**：去重、精简、归纳
4. **确认**：展示提炼内容，等待 Sam 确认后写入

## 适用内容

| 类型 | 章节 |
|------|------|
| 架构变更 | 架构层面 |
| 血泪教训 | 人机协同教训 |
| 工作流优化 | 记忆管理 |
| 里程碑 | 历史里程碑 |

## 检索长期记忆

```bash
grep "教训" MEMORY.md
grep "架构" MEMORY.md
```
