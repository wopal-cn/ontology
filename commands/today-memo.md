# 今日备忘 (Today Memo)

## 内容: $ARGUMENTS

## 流程

1. **提炼**：根据 `$ARGUMENTS` 或回顾会话，提取决策、进展、问题、待办
2. **写入**：追加到 `memory/YYYY-MM-DD.md`，格式：
   ```
   ### HH:MM 主题
   - [类型] 要点 1
   - [类型] 要点 2
   ```
3. **确认**：`📝 已记录`

## 类型标记

| 标记 | 用途 |
|------|------|
| `[决策]` | 架构/技术选型、重要结论 |
| `[里程碑]` | 完成的任务、达到的阶段 |
| `[教训]` | 踩过的坑、需要避免的 |
| `[TODO]` | 待办事项、需要跟进的 |
| `[关键事务]` | 需要记住的重要信息 |

## 触发场景

| 方式 | 示例 |
|------|------|
| `/today-memo` | 自动回顾会话，提炼记录 |
| `/today-memo [决策] 用 SQLite` | 记录具体决策 |

## 检索示例

```bash
grep "\[教训\]" memory/*.md      # 查找所有教训
grep "\[TODO\]" memory/*.md      # 查找所有待办
grep "\[决策\]" memory/2026-*.md # 查找某月的决策
```
