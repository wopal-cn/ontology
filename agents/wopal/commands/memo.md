# 快速记录 (Memo)

## 内容: `$ARGUMENTS`

## 操作步骤

1. **确定目标文件**：`memory/YYYY-MM-DD.md`（使用 `date '+%Y-%m-%d'` 获取日期）
2. **获取时间**：使用 `date '+%H:%M'` 获取当前时间
3. **直接追加**（无需读取文件）：
   ```bash
   cat >> memory/YYYY-MM-DD.md <<'EOF'

   ### HH:MM 主题
   - [类型] 内容
   EOF
   ```
4. **确认**：`📝 已记录`

## 类型标记

| 标记 | 用途 |
|------|------|
| `[知识]` | 隐性知识、非显而易见的事实 |
| `[经验]` | 操作技巧、有效做法 |
| `[教训]` | 踩坑记录、负向警示 |
| `[决策]` | 临时决策、进度记录 |
| `[TODO]` | 待办事项 |

## 日记格式

```
### HH:MM 主题
- [类型] 要点 1
- [类型] 要点 2
```
