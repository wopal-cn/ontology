# 今日备忘 (Today Memo)

## 内容: `$ARGUMENTS`

## 操作步骤

1. **确定目标文件**：`memory/YYYY-MM-DD.md`（使用 `date '+%Y-%m-%d'` 获取日期）
2. **提炼内容**：
   - 有 `$ARGUMENTS` → 直接记录
   - 无参数 → 提取会话关键信息（参见"类型标记"）
3. **直接追加**（⚠️ 无需读取文件，直接追加）：
   ```bash
   cat >> memory/YYYY-MM-DD.md <<'EOF'

   ### HH:MM 主题
   - [类型] 要点
   EOF
   ```
4. **确认**：`📝 已记录到今日备忘`

## 类型标记

| 标记 | 用途 |
|------|------|
| `[决策]` | 架构/技术选型、重要结论 |
| `[里程碑]` | 完成的任务、达到的阶段 |
| `[教训]` | 踩过的坑、需要避免的 |
| `[TODO]` | 待办事项、需要跟进的 |
| `[关键事务]` | 需要记住的重要信息 |

## 日记格式

```
### HH:MM 主题
- [类型] 要点 1
- [类型] 要点 2
```
